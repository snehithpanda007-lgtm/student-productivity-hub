# Student Productivity Hub

A React-based productivity dashboard for students.

## Features

- Add tasks
- Delete tasks
- Mark complete
- Category filtering
- Persistent local storage

## Tech Stack

- React
- Vite
- JavaScript
- CSS/Tailwind
